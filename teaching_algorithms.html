2. KNN Predictor

KNN là phương pháp suy luận dựa trên mẫu gần nhất. Thuật toán không xây dựng mô hình tham số cố định mà lưu trữ tập dữ liệu lịch sử, sau đó so sánh mẫu hiện tại với các mẫu quá khứ để suy ra kết quả dự đoán.

3. A* Pathfinding

A* là thuật toán tìm đường dựa trên đánh giá chi phí tích luỹ và heuristic. Mỗi đỉnh hoặc ô lưới được gán ba giá trị cơ bản gồm g, h và f = g + h, qua đó thuật toán lựa chọn hướng mở rộng hiệu quả nhất tại mỗi vòng lặp.

Dữ liệu vào: map grid + obstacle + start/goal Nguyên lý: g, h, f Đầu ra: waypoint path

4. Mô hình tích hợp ba thuật toán

Trong hệ AI hoàn chỉnh, các thuật toán không hoạt động riêng lẻ mà được tổ chức theo chuỗi xử lý. State Tree quyết định chiến lược tổng quát, A* xây dựng tuyến di chuyển, trong khi KNN hỗ trợ dự đoán vị trí tương lai của mục tiêu để nâng cao hiệu quả tác chiến.

Dữ liệu vào: world snapshot + map + memory Chuỗi xử lý: decision → path → prediction Đầu ra: route + aim + action
